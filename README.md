# Галерея для iPhone 🖼️

Веб-приложение галереи фотографий в стиле **Liquid Glass** (жидкое стекло) —
копия приложения «Фото» на iPhone как стартовая точка для дальнейшей кастомизации.

## Стек
- **Vite** + **React 18** + **TypeScript**
- Чистый CSS (`backdrop-filter` для эффекта стекла, без UI-библиотек)

## Возможности (тестовая версия)
- 📚 Медиатека: сетка фото с группировкой по датам
- 🔍 Сегменты «Годы / Месяцы / Все» меняют плотность сетки
- 🗂️ Альбомы (Избранное, Недавние, Все фото)
- 🔎 Поиск по подписям
- 🖼️ Полноэкранный просмотр фото (избранное, шторки, закрытие по Esc/тапу)
- 📱 Корпус iPhone на десктопе, полный экран на телефоне

## Запуск
```bash
npm install
npm run dev       # http://localhost:5173
```

С телефона в той же сети — открой адрес `Network`, который покажет Vite.

## Сборка
```bash
npm run build     # результат в dist/
npm run preview   # локальный предпросмотр сборки
```

## Структура
```
src/
  main.tsx            точка входа
  App.tsx             состояние и экраны (вкладки, зум, просмотр)
  styles.css          стили Liquid Glass
  icons.tsx           SVG-иконки
  data/photos.ts      тестовые данные фото
  components/
    StatusBar.tsx     статус-бар iOS
    TabBar.tsx        нижняя панель вкладок
    PhotoGrid.tsx     сетка фотографий
    PhotoViewer.tsx   полноэкранный просмотр
```

> Тестовые фото подгружаются с picsum.photos. Дальше можно заменить на свои.
